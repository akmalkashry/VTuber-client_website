## 🛠️ Tech Stack
This project is built using vanilla web standards with no external frameworks, keeping the build incredibly fast and lightweight.

* **Frontend:** Semantic HTML5, Advanced CSS3, and Vanilla JavaScript (ES6+).
* **IDE:** Engineered using **Visual Studio**.
